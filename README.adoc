Une application Spring Boot conçue pour recevoir des événements sous forme d'objets Avro via Kafka.

== Structure

Ce projet est construit comme un projet Maven basé sur le JDK 25 et Spring Boot 4.

Il utilise les technologies suivantes :

* **Spring for Apache Kafka** pour l'intégration avec Kafka.
* **Apache Avro** pour la définition des schémas d'événements.
* **Confluent Schema Registry** pour la gestion des schémas.
* **Liquibase** pour la gestion de la base de données.
* **Cucumber** pour les tests d'acceptation et la documentation vivante.

[NOTE]
====
Les répertoires qui démarrent par un point contiennent des configurations additionnelles :

* `.editorconfig` - configure les standards de formatage
* `.github` - configure le CICD
* `.idea` - configure IntelliJ
* `.mvn` - configure la toolchain Maven
====

== Cycle De Vie

|===
|Action |Commande

|Génération du code Avro |`./mvnw generate-sources`
|Construction |`./mvnw package`
|Documentation |`./mvnw verify`
|Démarrage |`./mvnw spring-boot:run`
|===
